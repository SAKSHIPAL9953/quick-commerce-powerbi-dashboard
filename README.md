# 📊 Quick Commerce Analytics Dashboard (Power BI)

## 🚀 Project Overview
This project presents an interactive Power BI dashboard designed to analyze operations, delivery performance, and customer behavior in a quick commerce environment (Blinkit/Zepto-style platforms).

---

## 📌 Dashboard Sections

### 1️⃣ Executive Summary
- Total Orders, Avg Delivery Time, Avg Order Value
- Orders by Company and Product Category
- On-time vs Delayed Orders

### 2️⃣ Operations Analysis
- Delivery Time vs Distance Analysis
- Company-wise Delivery Performance
- City-wise Delivery Delays
- Partner Rating Comparison

### 3️⃣ Customer & Revenue Insights
- Orders by Payment Method
- Average Order Value by Category
- Top 5 Categories by Orders
- Impact of Discounts on Order Value

---

## 🔍 Key Insights

- Delivery time increases significantly with distance  
- Certain cities show higher delivery delays  
- Performance varies across companies  
- UPI is the most preferred payment method  
- Orders are concentrated in top product categories  
- Discounts influence customer spending behavior
